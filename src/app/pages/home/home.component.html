Prerequisites

Ngb-dropdown uses Bootstrap for styling. If you don't have it already installed, install it with the command bellow.

npm i bootstrap --save

After installing bootstrap, you must inclue styles into you global syles.css/.scss file like bellow.

@import "~bootstrap/dist/css/bootstrap.css"

Or declare this styles into angular.json file, in the styels section.

{{importStylesSnippet}}

Installing

Ngb-dropdown is available at npm, if you have an existing application run the following command to download it to your project.

npm i @atheodosiou/ngb --save

Enabling animations (optional)

Ngb-dropdown utilize angular animations to improve the user experience, starting with Angular 4 animations have their own module so you need to import BrowserAnimationsModule to your application. If you prefer to disable animations globally, import NoopAnimationsModule instead.

npm install @angular/animations --save
